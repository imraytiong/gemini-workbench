# Gemini Workbench

A standardized, containerized hub for all Gemini CLI activities. Designed for developer efficiency, security, and portability.

## Getting Started

### Prerequisites
- Node.js (v18+)
- Podman (for container isolation)
- Git

### Installation
1. Clone this repository.
2. Run the automated setup script:
   ```bash
   bash setup-workbench.sh
   ```
3. Follow the CLI instructions to finalize your PATH configuration.

## Features
- **Isolated Sandboxes**: Full Ubuntu environment via Podman.
- **Portable Policies**: Workspace-integrated safety configurations.
- **Agentic Ready**: Designed for Gemini CLI automation workflows.
- **Open Source**: Licensed under Apache 2.0.
