# Project Scaffolder Skill

Automates the creation and configuration of new Gemini CLI projects with standardized files and Git settings.

## Instructions
1. **Initialize Project:** `node scripts/scaffold.cjs "<project-name>"`.
2. **Setup Git:** Initializes a new Git repository if one doesn't exist.
3. **Standards:** Automatically creates `GEMINI.md`, `TODO.md`, and `.gitignore`.
4. **Project Awareness:** Ensures the project is correctly configured for the `remember` extension.

## Workflow
Use this skill when the user wants to start a new project or standardize an existing one.
