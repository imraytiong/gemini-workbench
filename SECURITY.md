# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please report it immediately.

### Reporting Process
Please email your findings to [insert-email-or-security-contact] with the following details:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report within 48 hours and work with you to remediate the issue. Please do not publicly disclose the vulnerability until we have had an opportunity to address and patch it.
